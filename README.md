# المنصات الإلكترونية بكلية الشريعة والأنظمة

موقع ثابت يجمع روابط المنصات الإلكترونية الخاصة بكلية الشريعة والأنظمة بجامعة الطائف في صفحة واحدة متجاوبة مع الجوال واللابتوب.

## المنصات المضافة

- النشاط العلمي للأعضاء
- مؤشرات الأداء للكلية
- خطط ومقررات كلية الشريعة
- تقويم الفعاليات والمؤتمرات
- ساحة الاستطلاعات والرأي

## التشغيل المحلي

يمكن فتح `index.html` مباشرة في المتصفح، أو تشغيل خادم محلي بسيط:

```bash
python3 -m http.server 8000
```
