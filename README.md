# مركز عمليات كلية الشريعة والأنظمة

موقع ثابت يجمع روابط المنصات الإلكترونية الخاصة بكلية الشريعة والأنظمة بجامعة الطائف في واجهة تشبه لوحة معلومات موحدة. التصميم الحالي يقدم نموذجًا عمليًا لـ "مركز عمليات كلية الشريعة" مع شريط علوي موحد، شريط جانبي للأدوات، بحث ذكي، بطاقات متابعة، وتبويب مدمج للمنصات.

## المنصات

- النشاط العلمي للأعضاء
- مؤشرات الأداء للكلية
- خطط ومقررات كلية الشريعة
- تقويم الفعاليات والمؤتمرات
- ساحة الاستطلاعات والرأي

## ملامح الواجهة

- لوحة معلومات تعرض ملخصًا حيًا لكل منصة.
- بحث موحد يحاكي البحث عبر الأدوات الخمس.
- تبويب مدمج يفتح المنصة المحددة داخل الصفحة.
- شريط شخصي للمهام والأنشطة والروابط السريعة.
- تصميم متجاوب للجوال واللابتوب.

## التشغيل المحلي

يمكن فتح `index.html` مباشرة في المتصفح، أو تشغيل خادم محلي بسيط:

```bash
python3 -m http.server 8000
```
