# منصات كلية الشريعة والأنظمة

موقع ثابت بسيط يجمع روابط المنصات الإلكترونية الخاصة بكلية الشريعة والأنظمة بجامعة الطائف في صفحة واحدة متجاوبة مع الجوال واللابتوب.

## المنصات

- النشاط العلمي للأعضاء
- مؤشرات الأداء للكلية
- خطط ومقررات كلية الشريعة
- تقويم الفعاليات والمؤتمرات
- ساحة الاستطلاعات والرأي

## ملامح الواجهة

- واجهة بسيطة بدون صورة خلفية.
- خمس دوائر أنيقة لكل منصة بلون خاص.
- تأثيرات حركية ناعمة عند المرور (تكبير + توهج + دوران الأيقونة).
- تجاوب ذكي: 5 دوائر على اللابتوب، 3 ثم 2 على التابلت، 2 + 1 على الجوال.
- كل دائرة تفتح المنصة مباشرة في تبويب جديد.

## التشغيل المحلي

يمكن فتح `index.html` مباشرة في المتصفح، أو تشغيل خادم محلي بسيط:

```bash
python3 -m http.server 8000
```
